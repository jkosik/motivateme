# MotivateMe

Counter dApp on Ink blockchain with WalletConnect v2 support.

## Use Cases
- Just smile and time-based release
- Manual confirmation from sender
- Oracle integration

## Prerequisites

- Node.js (v18+)
- Go (v1.21+)

## Setup

```bash
npm install
# 1. Build frontend (run this after any changes to src/static/)
npm run build

# 2. Start Go server
cd src
go run main.go

# Visit http://localhost:8080
```

## Configuration

### Network Selection

Edit `src/static/index.html`:
```javascript
const ACTIVE_NETWORK = INK_SEPOLIA;  // or INK_MAINNET
```

### Contract Address

Edit `src/static/index.html`:
```javascript
const CONTRACT_ADDRESS = '0x...';
```

## Project Structure

```
.
├── src/
│   ├── static/          # Source files (you edit these)
│   │   └── index.html
│   └── main.go          # Go web server
├── dist/                # Built files (generated by npm build)
├── package.json         # npm dependencies
└── vite.config.js       # Build configuration
```

## Deployment

```bash
# Build frontend
npm run build

# Run Go server (serves files from dist/)
cd src
go run main.go
```


npm run dev
This starts a development server at http://localhost:5173
